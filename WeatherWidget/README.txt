To compile and run this program:

-Make sure to have javaFX installed and add the following arguments to the run configuration:
	--module-path "C:\JavaFX\javafx-sdk-11.0.2\lib" --add-modules javafx.controls,javafx.fxml
	(This assumes you have installed JavaFX in your C Drive in a folder named JavaFX)
	
-Download JSONObject jar and point to it
	You can get a copy @ https://jar-download.com/artifacts/org.json
	
-I used java jdk-15.0.1 as well

-This program uses an API call to openweathermap.org to get the weather data. You can get a free key by visiting their
website and creating an account.
